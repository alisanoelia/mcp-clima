# mcp-clima

### Instalation

```sh
git clone https://github.com/alisanoelia/mcp-clima
```

```sh
npm run build
```

### Claude or Client MCP

```json
"clima": {
    "command": "node",
    "args": [
        "/path/mcp-clima/build/index.js"
    ]
}
```

### Configuration API

```sh
export OPEN_WEATHER_MAP_KEY = "apikey"
```
